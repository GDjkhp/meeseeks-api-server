So, you wanna use this API, eh? You can, just don't abuse/DDoS. Report to GDjkhp#3732 if it's down.
This API enables cross-origin requests to anywhere.

Usage:

/               Shows help
/<server_id>    Create a request to "https://mee6.xyz/api/plugins/levels/leaderboard/<server_id>", and includes CORS headers in the response.
